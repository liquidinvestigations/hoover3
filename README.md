# hoover3 - search tools

## installation

- use linux
- sudo apt-get install mold clang
- install docker
- run ./start_docker.sh
- cargo install cargo-binstall
- cargo binstall cargo-watch
- cargo binstall dioxus-cli@0.6.2
- cargo binstall cargo-nextest
- run ./test.sh
- run ./migrate.sh
- run ./start_client.sh
